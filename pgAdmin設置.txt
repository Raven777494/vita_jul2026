:: 1. 設定 PostgreSQL 安裝路徑（媽媽幫你寫好，你只需確認路徑）

:: 2. 進入你解壓嘅 pgvector 資料夾（改成你嘅路徑）

\dx

確認仲係 vector 0.8.1！

第一步：在 psql 內強制註冊 ip_ops 運算子（最關鍵！）
開 psql -U postgres，貼以下指令（一行一行執行）：
SQL-- 強制重新載入 vector 定義，讓 ip_ops 運算子出現
DROP EXTENSION IF EXISTS vector CASCADE;
CREATE EXTENSION vector;

-- 手動註冊 ip_ops（Windows 18 必要步驟）
CREATE OPERATOR CLASS vector_ip_ops
DEFAULT FOR TYPE vector USING hnsw AS OPERATOR 1 <-> (vector, vector) FOR ORDER BY float_ops;

-- 驗證運算子已存在
SELECT opcname FROM pg_opclass WHERE opcname = 'vector_ip_ops';
